# Cord SDK Demo Apps

This repo contains opensource code for the demo apps on docs.cord.com and cord.com

See:

- [Dashboard](https://docs.cord.com/get-started/demo-apps/dashboard)
- [Document](https://docs.cord.com/get-started/demo-apps/document)
- [Video player](https://docs.cord.com/get-started/demo-apps/video-player)
- [Canvas](https://docs.cord.com/get-started/demo-apps/canvas-new)

The demo apps are built with Vite: just clone this repository, run `npm install`
in the root and within any app folder run `npm install` and `npm run dev`.
